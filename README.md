# datablock

## dependencies
- fsnotify
- install : go get -u github.com/fsnotify/fsnotify
- golang.org/x/sys 이것도 자동으로 설치됨.
- fsnotify v1.8.0, x/sys v0.26.0

## 개발 시작
- 먼저 사용자 입력 값은 최소한으로 받도록 한다. main 으로 일단 접근해서 향후 필요에 따라서 붙이는 방향으로 간다.
